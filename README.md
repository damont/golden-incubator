## For AI Assistants

**[Read AGENTS.md](AGENTS.md)** — Instructions for AI to guide clients through requirements gathering and architecture.

## Vision

A system where AI agents can take a client from idea → requirements → working MVP, with:
- Structured requirements gathering
- Pre-built project templates
- Automated environment setup
- Built-in testing & deployment
- Admin/support portals included

## Core Principle

**Template everything.** Every piece of the software building process should be a reusable template that AI can orchestrate:
- Project scaffolding templates
- Architecture patterns
- Deployment configurations
- Admin dashboard templates
- Support system templates
